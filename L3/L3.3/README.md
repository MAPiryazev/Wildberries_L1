# L3.3 Service

Микросервис для работы с древовидными комментариями.  
Реализован на **Go** с использованием фреймворка `github.com/wb-go/`

---

## Функционал

- Создание, получение, удаление комментариев.
- Поиск комментариев по тексту.
- Поддержка древовидной структуры комментариев без ограничений.
- Обработка ошибок через кастомные типы и `errors.Is`.
- Логирование HTTP-запросов через middleware.


## Запуск

1. Убедись, что PostgreSQL запущен.
2. В файле (`L3.3/cmd/server`) выполни:

```bash
go run server.go
```

3. Открой в браузере:  
   [http://localhost:<порт_из_.env>](http://localhost:<порт_из_.env>)
